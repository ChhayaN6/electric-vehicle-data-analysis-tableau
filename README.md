🚗 Electric Vehicle Population Data Analysis – Tableau Dashboard

📊 Project Overview



This project analyzes Electric Vehicle population data to uncover trends in EV adoption, manufacturer dominance, geographic distribution, and eligibility segmentation.



The dataset contains over 150,000 vehicle records.



🛠 Tools Used



Tableau (Data Visualization \& Dashboard Design)



Microsoft Excel (Data Cleaning \& Preprocessing)



🔍 Analysis Performed



EV growth trend by model year



State-wise vehicle distribution



Market share by manufacturer



Top vehicle models



BEV vs PHEV comparison



CAFV eligibility breakdown



Average electric range analysis



📈 Key Insights



BEVs dominate EV registrations



Significant growth observed post-2018



Tesla holds major market share



Geographic concentration visible in select states
